DiDi Traffic Data Analysis

In this project, you are given the traffic data of Chengdu within one day, containing the Ordering Information and the Route Information. Your job is to utilize the given data and the Online Map API, like Amap(http://lbs.amap.com/), to obtain some useful information from the data and display it on the map. JavaScript API for Amap is preferred.
